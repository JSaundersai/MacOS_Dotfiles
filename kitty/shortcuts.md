# Kitty Keyboard Shortcuts

## Clipboard
- **Cmd+C** - Copy to clipboard
- **Cmd+V** - Paste from clipboard

## Window Management
- **Cmd+Shift+Enter** - New window with current directory
- **Cmd+W** - Close window
- **Cmd+Shift+F** - Toggle fullscreen

## Tab Management
- **Cmd+T** - New tab with current directory
- **Cmd+Shift+W** - Close tab
- **Cmd+Shift+]** - Move tab forward
- **Cmd+Shift+[** - Move tab backward

## Tab Navigation
- **Cmd+1** - Go to tab 1
- **Cmd+2** - Go to tab 2
- **Cmd+3** - Go to tab 3
- **Cmd+4** - Go to tab 4
- **Cmd+5** - Go to tab 5
- **Cmd+6** - Go to tab 6
- **Cmd+7** - Go to tab 7
- **Cmd+8** - Go to tab 8
- **Cmd+9** - Go to tab 9
- **Cmd+0** - Go to tab 10

## Terminal Control
- **Cmd+K** - Clear terminal (scroll active)
- **Ctrl+Cmd+R** - Reload config

## Theme Switching
```bash
# List all themes
./switch-theme.sh

# Switch by name
./switch-theme.sh tokyonight_night

# Switch by number
./switch-theme.sh 1
```

## Transparency Toggle
```bash
./toggle-transparency.sh
```
Toggles between 80% opacity (glossy) and 100% opacity (opaque).
